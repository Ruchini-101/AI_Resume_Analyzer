🚀 AI Resume Analyzer

An AI-powered web application that analyzes resumes and provides instant feedback to help improve job readiness and ATS compatibility.

🌐 Live Demo: https://airesumeanalyzer-cv.streamlit.app/

📁 Repository: https://github.com/Ruchini-101/AI_Resume_Analyzer.git

📌 About the Project

The AI Resume Analyzer is a mini project built using Python and Streamlit. It helps users understand how well their resume aligns with job requirements by analyzing content, extracting key skills, and providing improvement suggestions.

This project demonstrates how AI/NLP techniques can be applied in real-world HR and recruitment scenarios.

✨ Features
📄 Upload and analyze resumes (PDF/Text)
🧠 Extract key skills and experience
📊 Basic ATS-style resume scoring
💡 Smart improvement suggestions
⚡ Fast and simple Streamlit interface
🛠️ Tech Stack
Python 🐍
Streamlit 🎈
NLP / AI Concepts 🤖
PDF/Text Processing Libraries
